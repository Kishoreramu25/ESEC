# Placement Navigator

A comprehensive Placement Management System standalone application.

## Project info

This project streamlines the campus placement process, offering dashboards for Placement Officers, HODs (Department Coordinators), and Management.

## How to run locally

**Requirements:** Node.js & npm installed.

```sh
# Step 1: Install dependencies
npm install

# Step 2: Start the development server
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (Backend)

## Deployment

You can deploy this project to Vercel, Netlify, or any static hosting service.
